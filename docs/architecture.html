Architecture Document

Layered Pattern

Controller (FastAPI Router) → Service (business rules) → Repository (DB operations) → PostgreSQL.

Components

  • API Gateway (FastAPI + Uvicorn)
  • Auth Component (JWT token issue + RBAC dependencies)
  • Request Management Component (state machine + SLA)
  • Audit Component
  • Export Component (PDF, CSV, Excel)
  • React Main UI
  • AI Agent UI (Streamlit)

Sequence

User authenticates, raises request, approver transitions status, fulfillment and closure recorded, exports generated on demand.
